- Experiencia con infraestructura como código (CDK o Terraform)
- Deseable: experiencia con Well-Architected Reviews

## Requisitos no técnicos
- Capacidad de interlocución con equipos técnicos del cliente (CTO, VP Ingeniería)
- Experiencia trabajando en proyectos con timelines ajustados
- Inglés intermedio o superior
