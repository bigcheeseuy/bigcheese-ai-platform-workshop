# Casos de Éxito de Referencia — BigCheese

## Caso 1: Plexus / BIOERIX — Arquitectura escalable para healthtech
**Sector:** Salud  
**Servicio:** Migration & Modernization + Optimization  

**Situación inicial:** Plexus operaba con infraestructura híbrida con problemas de escalabilidad en picos de demanda y alta latencia en el procesamiento de datos clínicos.

**Solución implementada:**
- Migración a arquitectura serverless en AWS (Lambda + API Gateway)
- Base de datos migrada a Amazon Aurora con réplicas de lectura
- Implementación de Auto Scaling Groups para cargas variables
- Monitoreo con CloudWatch y Datadog

**Resultados:**
- 40% de reducción en costos de infraestructura
- Latencia reducida de 800ms a 120ms en promedio
- Escalabilidad automática ante picos sin intervención manual
- Resolución de incidentes en menos de 15 minutos gracias a observabilidad

**Testimonio:** *"BigCheese ha sido un socio estratégico en el diseño de nuestras arquitecturas escalables, optimización de recursos y control de rendimiento."* — Martín Dotta, CCO, BIOERIX

## Caso 2: Apptim — Lanzamiento de producto comercial en AWS
**Sector:** SaaS / Tecnología  
**Servicio:** Development (Cloud Native) + Migration  

**Situación inicial:** Apptim necesitaba lanzar su producto principal al mercado rápidamente con una infraestructura confiable y de alto rendimiento desde el día uno.

**Solución implementada:**
- Arquitectura Cloud Native desde cero en AWS
- CI/CD con AWS CodePipeline y CodeDeploy
- Infraestructura como código con AWS CDK
- Ambientes de staging y producción aislados

**Resultados:**
- Time-to-market reducido en un 35%
- Infraestructura lista para escalar desde el lanzamiento
- Zero downtime deployments desde el primer día
- Costos operativos predecibles desde el inicio

**Testimonio:** *"Su conocimiento de AWS y su pasión por el rendimiento nos hacen sentir muy cómodos trabajando juntos como equipo."* — Fabian Baptista, CTO & Co-founder, Apptim

## Caso 3: Nuevo Siglo — Modernización de procesamiento de datos IoT
**Sector:** Industria / IoT  
**Servicio:** AI/ML Data + Migration & Modernization  

**Situación inicial:** Nuevo Siglo procesaba datos de dispositivos IoT con pipelines legacy que no podían escalar ni incorporar análisis en tiempo real.

**Solución implementada:**
- Migración de pipelines a AWS IoT Core + Kinesis Data Streams
- Procesamiento en tiempo real con AWS Lambda
- Data Lake en S3 con AWS Glue para ETL
- Dashboards analíticos con Amazon QuickSight

**Resultados:**
- Procesamiento de datos en tiempo real (antes era batch con 4hs de delay)
- Capacidad de ingesta escalada 10x sin cambios de infraestructura
- Nuevos insights de negocio disponibles para el equipo comercial
- Reducción del 25% en costos de procesamiento

**Testimonio:** *"BigCheese nos ayudó a pensar de otra manera, buscando aprovechar al máximo las herramientas disponibles."* — Mario De Oliveira, CTO, Nuevo Siglo
